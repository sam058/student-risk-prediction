# Student Dropout Risk Prediction System

This project predicts at-risk students using learning behavior analysis and machine learning.

## Key Features

- Behavioral clustering of students based on LMS interaction
- Early prediction of at-risk students
- Cluster-aware weighted XGBoost model
- Interactive Streamlit dashboard for risk analysis

## Tech Stack

- Python
- Scikit-learn
- XGBoost
- Pandas, NumPy
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py

```
